# Contributing

Thanks for taking the time to contribute.

## Ways to contribute
- Report bugs or suggest improvements via issues.
- Submit pull requests for fixes or new features.
- Improve documentation and examples.

## Development setup
- Go 1.20+ recommended.
- Python 3.9+ for plugins.

## Pull request guidelines
- Keep changes focused and scoped.
- Include tests or manual verification notes when applicable.
- Run `go test ./...` when touching Go code.
- Keep CLI behavior backward compatible where possible.

## Code style
- Go: `gofmt` on changed files.
- Python: follow PEP8 and keep plugin output stable.

## Security
If you believe you have found a security issue, please report it privately and do not open a public issue.
